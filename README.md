# Walidator Schematu JSON

Prosty walidator schematu JSON napisany w Javie.

## Jak to działa?

Ten projekt dostarcza klasę `JsonSchemaValidator`, która potrafi zweryfikować, czy dany obiekt JSON jest zgodny z określonym schematem JSON.

## Jak uruchomić?

Ten projekt nie jest skonfigurowany z narzędziem do budowania, takim jak Maven czy Gradle. Aby go uruchomić, należy otworzyć go w środowisku programistycznym (IDE), takim jak IntelliJ IDEA, a następnie uruchomić odpowiednią klasę z metodą `main`.

## Jak uruchomić testy?

Testy jednostkowe znajdują się w `src/test/java/validator/JsonSchemaValidatorTest.java`. Aby je uruchomić:

1.  Otwórz projekt w IntelliJ IDEA.
2.  Przejdź do pliku `JsonSchemaValidatorTest.java`.
3.  Kliknij prawym przyciskiem myszy na klasę `JsonSchemaValidatorTest` i wybierz "Run 'JsonSchemaValidatorTest'".

Spowoduje to uruchomienie wszystkich testów w tej klasie i wyświetlenie wyników w IDE.
